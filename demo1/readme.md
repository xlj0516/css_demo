###主要点是
------------
1. div:nth-of-type()使用伪类选择器来对不同的div和<input>写样式
2. input[type="radio"]：属性选择器
3. input[type="radio"]:checked+div :属性选择器，伪类选择器，相邻选择器
4. transform:translateX() 指定对象（水平方向）的平移
5. transition:<' transition-property ',' transition-duration ',' transition-timing-function ',' transition-delay '>过渡
